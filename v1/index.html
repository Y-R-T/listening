使用说明

0. 这里是本项目的 GitHub 仓库，欢迎 Star 和 Fork！感谢支持！当前使用版本为V1

1. 点击 "上传新听力" 或从列表中选择一篇听力 (若从列表选择，可能需重新上传文件以播放)。

2. 键盘快捷键：

  • 播放/暂停: Space (空格键)
  • 设置记录点: Ctrl+Shift+B (Win/Linux) / Cmd+Shift+B (Mac)
  • 回到上个记录点: Ctrl+Shift+P (Win/Linux) / Cmd+Shift+P (Mac)

3. 鼠标快捷模式：

  • 点击 "🖱️ 进入快捷模式" 按钮激活。
  • 激活后，在播放器区域内（非按钮上）：左键点击 = 上一记录点, 右键点击 = 设置记录点。
  • 再次点击按钮退出该模式。

4. 进度条上会标记您的记录点 (●) 以及已精听的片段 (高亮区域)。

5. 完成整篇听力后，会有礼花庆祝！
